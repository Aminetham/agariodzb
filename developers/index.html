Quick explanation on how to make video skins


https://legendmod.ml/developers/videoskins.html



1. Make a github account
2. Make a repository with exactly same + .github.io
example jimboy3100 -> jimboy3100.github.io

3. Upload a video (.mp4 .webm .ogv) on github
The destination is:
example:
jimboy3100/jimboy3100.github.io (account: full repo address: jimboy3100/jimboy3100.github.io)
jimboy3100.github.io/banners/testvideodyinglight.mp4 (the video i have)
https:// + jimboy3100.github.io/banners/testvideodyinglight.mp43
https://jimboy3100.github.io/banners/testvideodyinglight.mp4

"banner" is a folder for me, so you it will be https://user.github.io/banners/testvideodyinglight.mp4
if your account is: user and your repository was name user.github.io

Use it as SKIN URL the same way you do for imgur icons

*To make a folder on github "create a new file", leave it blank and name it /yourfoldername/yourblackfilename.html
**Needs 5-10 minutes to load as "Ready to use URL"
***In rare occasions that one video does not work properly, use another video
**** On my own github i changed the destination "https://jimboy3100.github.io/" to "https://legendmod.ml/", example https://legendmod.ml/banners/testvideodyinglight.mp4
